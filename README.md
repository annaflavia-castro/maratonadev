# maratonadev
Project for donates blood, developed at MaratonaDev RocketSeat's with HTML, CSS and JavaScript.

**Dependencies**
---
- NodeJS
- NPM or Yarn
- PostgreSQL
- Express
- Nodemoon

**Install**
---
`https://github.com/annaflavia-castro/maratonadev.git`

`yarn install`

`yarn start`
